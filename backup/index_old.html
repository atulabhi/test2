What is OpenEBS ?

Cloud Native Storage

Storage is delivered as containers for containers.

Intent Inherited

Application intent is inherited in yaml files, which enables easy automation of your DevOps.

Granular Control

Granular control on your storage needs for your stateful applications.

Avoid cloud lock-in

Applications are now movable across clouds.

Integrated with Kubernetes

Integrates nicely into Kubernetes for provisioning, application-storage connectivity.

Written in GO

Open source friendly. OpenEBS is the only block storage open source stack that is written in GO.
