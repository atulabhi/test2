PROJECT UPDATES

OpenEBS 0.3

Hyper-converged storage for stateful applications on Kubernetes platforms. 
  • Released OpenEBS 0.3 - OpenEBS Storage Containers (VSMs) are completely orchestrated and managed via Kubernetes. OpenEBS 0.3, fully integrates into Kubernetes, allowing you to use OpenEBS storage on your existing Kubernetes. Read More
  • Easily deployable on Jupyter Notebooks- With the new release, OpenEBS data volumes, can adapt to the storage demands (auto-expansion of storage, temporary workspace required by the analytical operations, generating and sharing read-only copies of data to multiple notebooks).
