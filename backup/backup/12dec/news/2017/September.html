OpenEBS is celebrating being an open source enthusiast by participating in Hacktoberfest 2017.We'd love to get your contributions too. So visit our Hacktoberfest page and start hacking away at OpenEBS, an open source project written in Go.

OpenEBS v0.4 is released!
 
 OpenEBS just announced the release of OpenEBS v0.4. Sticking true to our promise, we are continuing on the path of making the product easier to use. This is a developer friendly release where a developer can make the minikube deployment OpenEBS enabled very quickly.

PROJECT UPDATES

September was an exciting month for OpenEBS at multiple levels:

  • We were present at several DevOps focused conferences both in India and the US. The value proposition that OpenEBS brings to the Stateful Applications resonated well with the Developers and Operations personnel grappling with issues around running Stateful applications with connected storage.
  • There were recurring requests from many of the Developers we met during the conferences to have OpenEBS made available for "minikube" setup and thats just what we have. Please check out our guide for running OpenEBS with minikube.
  • OpenEBS v0.4.0 has been released with significant improvements to documentation and functionality to manage OpenEBS volume snapshots. This release marks a major milestone by making OpenEBS a viable alternative to other open source container native storage for kubernetes clusters.
